## [Installation](https://heejung-gjt.github.io/django-01)

#### 가상환경 세팅하기

패키지 간의 버전차이, 여러 충돌을 방지하기 위해 가상환경을 구축한다.
가상환경이 성공적으로 설정되면 터미널안에 (venv_django) 가 생성된다.
 

```bash
$ python -m venv venv_django
$ source venv_django/Scripts/activate
$ pip install django
```
